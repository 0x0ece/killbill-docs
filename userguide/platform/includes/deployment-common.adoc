==== Download the latest version of Kill Bill server

Releases are available in Maven Central. Among other packaging formats, we provide a war with all dependencies. You can get the latest version http://killbill.io/downloads/[here] (download the *war* artifact).

==== Setup the database

Note: currently, only MySQL is supported for multi-nodes deployments.

Create the database killbill will use and apply the latest DDL schema, which can be found http://killbill.io/downloads/[here].
